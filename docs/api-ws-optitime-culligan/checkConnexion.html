Command checkConnexion       ( checkConnexionResponse )

Description
2 modes.
1 : ffaSession est pass&ecute; en paramètre. on check la sessin et on va recherche l'id du job

2 : login / password passé en paramètre . se connecte au connecteur et teste l'existence de la DS et du client selon le login / password saisi par l'internaute. si la DS n'est pas active la connexion est refusée.

Dans les 2 ac, on vérifie que le lien est encore actif'
