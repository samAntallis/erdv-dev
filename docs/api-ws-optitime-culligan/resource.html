Object resource

Description
A Resource is a resource that can perform a schedule item. It is defined by a user id and a name, and a set of abilities that are matched against profiles and schedule items required abilities to determine whether it can be assigned to the schedule item (among other constraints, such as unavailabilities and quotas).
A Resource is mapped in OTS as a resource. Its abilities are mapped as resource properties. Annex data are mapped as properties of the resource card.
