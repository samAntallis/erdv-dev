General float number format

General float number format is [+|-]9999999[.9999999][E[+|-]999]

exemple of valid numbers :
   52
   -52
   -52.123
   -52.123E-234
note that the sum of the number of digits of the interger part and the decimal part shoud be less or equal to 14 digits. the exponential part shoud be between -300 and +300
